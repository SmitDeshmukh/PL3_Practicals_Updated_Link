Online Assignment Portal For Government Polytechnic Pune

This Portal is aimed at downloading and uploading online assignments for students; with each assignment having information about the instructions, description, deadline, and submission details. The main goal of this Project is to design and implement online assignment submission they would be able to evaluate assignments automatically. The system provides an interface for testing assignments such that they can be plugged in by the teachers.The most obvious advantage offered by online assignment submission is that it offers faster transmission of assignments than using traditional way by using online system. The interface will help teachers, so save the time and cost for teachers by enabling them to put up a fast response for students as well as increasing the quality of the feedback provided to students.
Tech Stack : PHP,HTML,CSS,JS,MySql.
